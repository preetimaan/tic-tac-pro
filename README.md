## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Controls

- **Click** on a cell to place your piece
- **Click + Drag** to rotate the camera
- **Scroll** to zoom in/out
- **Pan** to move the camera
